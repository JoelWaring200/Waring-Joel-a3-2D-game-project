# Waring-Joel-a3-2D-game-project

## Rules
W or Space is to jump <br />
A or <- is to go left <br />
D or -> is to go right <br />

## Goal
Dodge attacks and wait for an opportunity to attack or heal

## Story
Three warriors stand guarding the gate, <br />
First fights by habit, yields if shown mercy, <br />
Second seeks honor, spared with kindness, <br />
Third acts from fear, surrenders if trusted. <br />
All can be spared, All can be killed <br />
